# 🧠 Ejercicios de Lógica en Java

Este repositorio contiene una colección de **ejercicios de lógica de programación desarrollados en Java**, organizados por categorías y enfocados en demostrar el dominio de **pensamiento lógico**, **manejo de estructuras de datos**, y **buenas prácticas en Programación Orientada a Objetos (POO)**.

El objetivo principal de este proyecto es **mostrar habilidades técnicas aplicadas**, más allá de soluciones teóricas, simulando escenarios comunes que enfrenta un desarrollador Java en contextos reales.

---

## 🎯 Objetivo del Proyecto

- Fortalecer la **lógica de programación**
- Aplicar conceptos fundamentales y avanzados de **Java**
- Resolver problemas de forma clara, eficiente y mantenible
- Mostrar capacidad de **análisis, organización y estructuración del código**
- Servir como **portafolio técnico** para procesos de selección

---

## 🧩 Contenidos y Conceptos Aplicados

Los ejercicios están diseñados para cubrir los siguientes conceptos clave:

### 🔹 Fundamentos de Java
- Variables primitivas y tipos de referencia
- Métodos y retorno de valores
- Uso adecuado de `static`
- Validaciones y control de errores

### 🔹 Estructuras de Control
- Condicionales (`if`, `else`)
- Bucles (`for`, `while`)
- Bucles anidados
- Control de flujo

### 🔹 Estructuras de Datos
- Arreglos unidimensionales
- Matrices bidimensionales
- Recorridos y transformaciones de matrices
- Manipulación de datos numéricos y textuales

### 🔹 Lógica Algorítmica
- Inversión de arreglos
- Eliminación de duplicados
- Identificación de valores únicos
- Cálculo de promedios
- Rotaciones de arreglos
- Matriz identidad
- Transpuesta de matrices
- Suma de matrices
- Validaciones de palíndromos

### 🔹 Buenas Prácticas
- Código legible y ordenado
- Separación de responsabilidades
- Nombres descriptivos de métodos y variables
- Organización por paquetes
- Enfoque en mantenibilidad
